# 🖼️ Insanely Expensive JPEGs (NFT Website)

A simple front-end project built to practice **essential CSS concepts** with a focus on writing **clean, reusable, and non-repetitive code**.

---

## 🎯 Purpose of the Project

This project was created with a clear goal:

* Learn and apply **core CSS fundamentals**
* Avoid unnecessary repetition in styling
* Structure code in a **clean and maintainable way**

Instead of repeating styles, shared classes and grouped selectors were used to keep the CSS efficient and readable.

---

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox, grouped selectors, reusable classes)

## ✨ Key CSS Concepts Practiced

* Reusable classes (e.g., `.container`, `.link-btn`)
* Grouped selectors to reduce repetition
* Consistent spacing system
* Flexbox layout
* Hover and active states
* Clean typography styling

---

## 🧠 What I Learned

This project reinforced an important principle:

> Good CSS is not about writing more — it’s about writing smarter.

* Reusability improves scalability
* Grouping reduces duplication
* Clean structure makes code easier to maintain
